Et si on passait à VS Code ? Notre option qui fait toute la différence.

Claude.ai impose le modèle Anthropic. VS Code est l'environnement du développeur. Notre ambition est qu'il soit aussi le vôtre. VS CODE permet de configurer librement le modèle utilisé : Anthropic, Mistral, Google, Meta, ou tout fournisseur proposant une API compatible. Si un autre modèle s'avère plus performant, mieux adapté à votre pratique ou plus éthique, vous saurez basculer sans perdre votre environnement de travail. C'est la seule interface qui garantit cette indépendance.

VSCODE permet également un travail plus fluide avec l'IA, est nettement moins énergivore en tokens, et les modèles y sont plus performants.L'interface a quelque-chose d'effrayant pour le juriste. Avec Nomos, ce sentiment devient un lointain souvenir.

II.Vos nouvelles pratiques

Trois pratiques fondamentales à acquérir.

P · I — Méthode

Construire son cadre méthodologique.

Compétences spécialisées, templates d'instructions permanentes par type de dossier : l'IA doit travailler selon votre méthode, pas la sienne. Ce que vous construisez pendant la formation, vous l'utilisez le lendemain.

P · II — Confidentialité

Configurer une confidentialité native.

Les données client ne transitent jamais en clair. L'anonymisation est intégrée au flux de travail, avant tout traitement, conformément aux recommandations déontologiques en vigueur.

P · III — Sources

Permettre à l'IA d'interroger les sources primaires.

Légifrance, CEDH, EUR-Lex, CURIA, HUDOC, BOFiP, Ariane Web : les bases publiques officielles, en accès direct. Les requêtes interrogent les sources en temps réel, sans approximation vectorielle. Pas d'hallucination. Et avec un bon skill, une IA qui sait comprendre une source.
